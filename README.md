# clef

An opinionated, Rails-inspired full-stack web framework for Clojure.

**Status: pre-alpha, draft mode.** Not usable yet. Public for development transparency; the install path is gated.

## Install (maintainer only, for now)

```
brew tap kirahowe/clef
brew install clef
```

The tap repo is private during the v0 phase; only the maintainer can install.
